# homebrew-vervain

Homebrew tap for [Vervain](https://github.com/yakupbulbul/Vervain).

## Install

```bash
brew tap yakupbulbul/vervain
brew install vervain
```

## What is Vervain?

A free, open-source macOS cleaner that respects your files and your privacy. Learn more at [vervain.app](https://vervain.app).
