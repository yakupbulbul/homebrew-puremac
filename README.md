# homebrew-puremac

Homebrew tap for [PureMac](https://github.com/yakupbulbul/PureMac).

## Install

```bash
brew install yakupbulbul/puremac/puremac
```

Or tap first:

```bash
brew tap yakupbulbul/puremac
brew install puremac
```

## What is PureMac?

A free, open-source macOS cleaner that respects your files and your privacy. Learn more at [puremac.app](https://puremac.app).
